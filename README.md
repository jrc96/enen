# enen v0.0.2 -- master(f19751d)
a game framework based on gamelib

## 启动命令(参照config/server.json配置文件)

- center服务：enen center
- gate服务：enen gate
- game服务：enen game
- gate1服务：enen gate -n=gate1
- game1服务：enen game -n=game1
